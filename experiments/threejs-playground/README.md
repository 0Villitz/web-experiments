# threejs-playground
## About
This is a simple test that renders a cube in the browser using threejs via CDN, and that is all. The goal was to learn the basics of threejs and have a working project to reference and study.

## Getting Started
- You must serve this to a browser using a server - simply opening index.html will not work. To do so I installed the [Live Server VSCode extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer), which allows me to right-click index.html and choose "Open with Live Server".

- If successful, you should see a green, rotating cube illuminated by one directional light. There is nothing else in this demo, just a rotating cube.
