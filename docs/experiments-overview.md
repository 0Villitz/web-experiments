# Experiments Overview

This document provides a high-level overview of the experiments included in this repository.

---

## Experiment Organization

Each experiment is stored in:

```
experiments/<experiment-name>/
```

Example structure:

```
experiments/
  threejs-playground/
    index.html
    main.js
    assets/
    README.md
```

## Current Experiments
